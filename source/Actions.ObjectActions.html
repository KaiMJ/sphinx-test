add_object(name, location, rotation, scale, properties, mesh_path=None, material_path=None, physics=False)

Add an object to the scene.

Parameters:

  • name – Name of the object.

  • location – Location of the object as a dictionary {‘x’: value, ‘y’: value, ‘z’: value}.

  • rotation – Rotation of the object as a dictionary {‘pitch’: value, ‘yaw’: value, ‘roll’: value}.

  • scale – Scale of the object as a dictionary {‘x’: value, ‘y’: value, ‘z’: value}.

  • properties – Dictionary of properties to set on the object.

  • mesh_path – Optional path to the mesh asset.

  • material_path – Optional path to the material asset.

  • physics – Boolean indicating whether physics should be enabled.

Returns:

ID of the added object.
